## 星体运动模拟

>### 使用方法

`yarn install`

`yarn start`

[在线预览](https://public.zkytech.top/stars/index.html)

>### 相关技术
2D模型的绘制就是单纯的canvas绘图

3D模型的绘制使用的是threejs，其本身是对WebGL的一个封装。
